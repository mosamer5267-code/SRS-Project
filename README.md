## Overview
CampusCare is a mobile application that improves facility management in a university campus by allowing users to report maintenance issues and track their status in real time.

## Features
- User authentication (signup, login, logout)  
- Issue reporting with location, photo, category, and description  
- Issue tracking with status updates (Pending, In Progress, Resolved)  
- Facility manager dashboard for assigning and managing issues  
- Worker interface for handling assigned tasks  

## Technology Stack
- Frontend: React Native  
- Backend: Node.js  
- Database: MongoDB  
- API: RESTful APIs  

## Key Requirements
- Role-based access control  
- Real-time issue tracking  
- Fast performance (under 3 seconds response time)  
- Support for multiple users
